# Minimums chosen so that the whole stack supports numpy 2.x.
# Historical context : scipy < 1.13, mne < 1.7 and scikit-learn < 1.4 still
# pin numpy < 2.0 transitively. On Google Colab the base image ships numpy
# 2.x (required by jax, opencv, shap, tifffile, etc.), so if we let pip pick
# an old scipy/mne it silently downgrades numpy and breaks the kernel state.
# The src/ code itself has been audited for numpy 2.0 breaking changes — no
# np.int/np.float/np.bool legacy aliases, no removed stable APIs.
mne>=1.7
numpy>=1.24
scipy>=1.13
scikit-learn>=1.4
pandas>=2.1
matplotlib>=3.8

# Demo dependencies are also useful for local validation of the dashboard.
streamlit>=1.30
plotly>=5.18
