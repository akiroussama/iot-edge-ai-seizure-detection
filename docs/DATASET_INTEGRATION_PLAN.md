# Dataset Integration Plan

This project must not make clinical claims until the public datasets are parsed, labels are audited, and leakage checks pass.

## Canonical Tables

Every dataset must produce:

- `metadata.parquet`: `patient_id`, `recording_id`, optional `center_id`, `source_dataset`.
- `events.parquet`: `patient_id`, `recording_id`, `seizure_start`, `seizure_end`, optional `seizure_type`, `center_id`, `source_dataset`.
- `recordings.parquet`: `patient_id`, `recording_id`, `recording_start`, `recording_end`, optional `center_id`, `source_dataset`.
- `windows.parquet`: generated by `scripts/make_windows.py`; do not hand-edit.
- `modality_availability.parquet`: `patient_id`, `recording_id`, `modality`, `available`, `sampling_rate`, `channel_count`, `notes`.

## SeizeIT2

Initial parser target: BIDS/OpenNeuro-like layout with `*_events.tsv` files.

Supported v0.1 metadata path:

```bash
uv run python scripts/prepare_seizeit2.py \
  --raw-dir data/raw/seizeit2 \
  --processed-dir data/processed/seizeit2 \
  --inspect-only
```

Then parse supported event files:

```bash
uv run python scripts/prepare_seizeit2.py \
  --raw-dir data/raw/seizeit2 \
  --processed-dir data/processed/seizeit2
```

Manual checkpoint: inspect 5-10 seizure timelines against the original annotations and verify event times are absolute recording times.

## My Seizure Gauge

Initial parser target: root `seizure_times.csv` plus per-patient ZIP files containing HR and steps streams.

```bash
uv run python scripts/prepare_msg.py \
  --raw-dir data/raw/msg \
  --processed-dir data/processed/msg \
  --inspect-only
```

Then parse supported event and modality manifest files:

```bash
uv run python scripts/prepare_msg.py \
  --raw-dir data/raw/msg \
  --processed-dir data/processed/msg
```

Manual checkpoint: confirm seizure timestamps are aligned to the wearable stream timezone and chronic EEG annotation timezone.

## Windowing And Labels

Generate windows from a verified `recordings.parquet`:

```bash
uv run python scripts/make_windows.py \
  --recordings data/processed/seizeit2/recordings.parquet \
  --out data/processed/seizeit2/windows.parquet \
  --window-duration 2min \
  --stride 30s
```

Generate SPH/SOP labels:

```bash
uv run python scripts/label_windows.py \
  --windows data/processed/seizeit2/windows.parquet \
  --events data/processed/seizeit2/events.parquet \
  --output data/processed/seizeit2/labels_sph5_sop30.parquet \
  --sph-minutes 5 \
  --sop-minutes 30 \
  --postictal-exclusion-minutes 60
```

Do not start A100 training until the label audit and leakage audit are clean.
